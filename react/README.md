# React

React projects for Sigma embedded analytics and custom UI components.

## Suggested layout

```
react/
└── <project-name>/   # One folder per app or component set
```

## Getting started

```bash
npx create-react-app my-sigma-app
cd my-sigma-app
npm install @sigmacomputing/plugin
npm start
```

## Useful packages

- [`@sigmacomputing/plugin`](https://www.npmjs.com/package/@sigmacomputing/plugin) — Sigma Plugin SDK
- [`@sigmacomputing/embed-sdk`](https://www.npmjs.com/package/@sigmacomputing/embed-sdk) — Sigma Embed SDK
