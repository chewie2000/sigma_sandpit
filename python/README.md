# Python

Python scripts, utilities, and notebooks for Sigma, Databricks, and Snowflake projects.

## Suggested layout

```
python/
├── databricks/   # Databricks notebooks and jobs
├── snowflake/    # Snowflake connectors and utilities
└── sigma/        # Sigma API integrations
```

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
